Community

Weld is an open-source project. Contributions and participation is welcome.

User forum

The Weld user forum is intended for developers who use Weld to build their applications.

IRC Chat

The #weld-dev IRC channel hosted on irc.freenode.net focuses on Weld development discussion. The conversation is logged and the log is available at transcripts.jboss.org

Mailing list

Discussions related to development of Weld also takes place in the weld-dev mailing list . An archive for the mailing list is available at lists.jboss.org

Source code repository

Weld uses Git for source code version control. All the source code is hosted on GitHub

We follow the Integration Manager Workflow for managing changes to the Weld source code. If you are new to using GitHub we recommend that you read this tutorial.

Reporting a bug

Bug reports and feature requests can be reported in the project JIRA

Contributors

  • List of Weld contributors

If you are interested in becoming a contributor, check the list of open issues awaiting contributions or let us know on the developer mailing list.
